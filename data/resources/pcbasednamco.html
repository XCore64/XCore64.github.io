Namco PC based System Boards

This page contains specifications of Namco Arcade System Boards that are using PC hardware. If you're curious on what specifications that one of the arcade system boards is used or if you're trying to match or exceed the specifications from your own, click on one of the systems below.

Namco System N2

This is an arcade system board based on a Debian based Linux OS. Uses an AMD CPU with a NVIDIA GPU. The games used on this system are developed with OpenGL API.

View details »

Namco System ES1

This is an arcade system board based on arcadelinux, a Debian 4.0 based Linux OS. Uses an Intel Core 2 Duo CPU with a NVIDIA GeForce 9600 GT.

View details »

Namco System ES3

This is an arcade system board that is running 32-bit or 64-bit using Windows Embedded 7 (Windows 7) depending on the game. It uses an Intel Core 2 Duo CPU with a NVIDIA GeForce GTX 680. The games used on this system are developed with Direct3D 11 API.
