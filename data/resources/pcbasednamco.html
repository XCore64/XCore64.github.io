Namco PC System Boards

This page contains specifications of Namco Arcade System Boards that are using PC hardware. If you're curious on what specifications that one of the arcade system boards is used or if you're trying to match or exceed the specifications from your own, click on one of the systems below.

Namco System N2

This is an arcade system board based on a Debian Linux OS. It uses an AMD Athlon 64 CPU with a NVIDIA GeForce GPU. The games used on this system are developed with NVIDIA's Cg Shading Language. This is a competitor of Sega's Lindbergh system.

View details »

Namco System ES1

This is an arcade system board based on arcadelinux, a Debian based Linux OS. It uses an Intel Core 2 Duo CPU with a NVIDIA GeForce 9600 GT. The games used on this system are developed with NVIDIA Cg Shading Language. There is an A2 revision of the ES1 for overseas arcade content.

View details »

Namco System ES3

This is an arcade system board that is running 32-bit/64-bit using Windows Embedded 7 (Windows 7) depending on the game. It uses an Intel Core 2 Duo CPU with a NVIDIA GeForce GTX 680. The games used on this system are developed with Direct3D 11 API. There are three revisions: A, B & X.
